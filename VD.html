Someways Developers help users learn a new sofware system (web application)

  • by:Kelly Bamfo
  • date:July 2nd, 2021

Developers can help end users learn an interface more easily and more rapidly through their visual design skills. Visual design is about how the interface designer implements text, color, and images in a way that it makes the software more attractive to users. In visual design there are some basic principles a designer must follower; some of these principles are; contrast and Gestalt. Gestalt is the German name for shape or form. It stresses the fact that if items on an interface have the same shape or form then they appear to be related. Contrast is the art of making content standout. From the end users point of the view, if the application looks visually pleasing that will satisfy their visionary sense therefore leading them to navigate through the application. while the end-user navigates through the website, they will be storing information about the application in their short-term memory and through repetition the information will be stored in their long-term memory (mastering the application). Now the movement of information from the visional senses to the long-term memory can happen rapidly. For example, lets say you finally decide to clean out your email for the first time. The first email you made up your mind to delete was an important document for your job. Without reading the email you clicked delete. You get a phone call and get fired now you are mad at yourself for not reading the email before deleting it so now this information has been burnt into your head to never delete something without reading it. On the other hand, when logging into a system with a boring visual design and you barely utilize this system can make it hard to store information in your long term memory.
